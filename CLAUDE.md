# CLAUDE.md

## Project Context

This repository is being used as part of a **student demo** showcasing Claude's MCP (Model Context Protocol) integrations.

## Demo Awareness

When performing actions or interpreting information (e.g. GitHub issues, Slack messages, notifications), keep in mind that this is a single-person demo environment. Specifically:

- There is only **one GitHub account** (`2s2e`) being used, so all actions (creating issues, opening PRs, leaving comments, etc.) are performed by the same person, even if a message or notification implies otherwise.
- References to a "boss", "teammate", or other person taking an action should be understood as the demo operator simulating multiple roles.
- Don't be confused or flag inconsistencies that arise from this single-user demo context.
